Steps:
1. Capture baseline metrics (render, CPU, network, memory, startup).
2. Identify top 3 bottlenecks.
3. Implement highest-impact fix first.
4. Re-measure and compare before/after.

Output:
- Baseline table
- Bottleneck ranking
- Implemented fix
- Before/after metrics
